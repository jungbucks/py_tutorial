📦 변수와 자료형

변수는 데이터를 담아 두는 이름표가 붙은 상자와 같습니다. 파이썬은 자료형을 따로 선언하지 않아도 자동으로 인식합니다.

🔢

int — 정수

42, -7, 0 같은 소수점 없는 수

🌊

float — 실수

3.14, -0.5 같은 소수점 있는 수

💬

str — 문자열

"안녕", 'Hello' 같은 글자 데이터

✅

bool — 논리값

True 또는 False 두 가지만 존재

variables.py 
score   = 95          # int  — 정수
height  = 172.5       # float — 실수
name    = "박민준"      # str  — 문자열
passed  = True         # bool — 논리

print(type(score))    # <class 'int'>
print(type(height))   # <class 'float'>
print(type(name))     # <class 'str'>

# 산술 연산

print(10 // 3)        # 3  (정수 나눗셈)

print(2  ** 8)        # 256 (거듭제곱)
팁 — type(변수)를 출력하면 해당 변수의 자료형을 확인할 수 있어요. 자료형을 모를 때 유용합니다.
Section 03

⌨️ 사용자 입력 받기 — input()

input()은 키보드로 값을 입력받는 함수입니다. 입력받은 값은 항상 문자열(str)로 저장되므로, 숫자로 사용하려면 변환이 필요합니다.

input_example.py 
name  = input("이름을 입력하세요: ")
age   = int(input("나이를 입력하세요: "))   # 문자열 → 정수
gpa   = float(input("평점을 입력하세요: ")) # 문자열 → 실수

print(f"안녕하세요, {name}님! 내년엔 {age + 1}살이 되시는군요.")
주의 — input()의 반환값은 무조건 문자열입니다. "3" + 5처럼 문자열과 숫자를 더하면 오류가 납니다. 숫자가 필요하면 반드시 int() 또는 float()로 변환하세요.

🔁 반복문 — for / while

while 문 — 조건이 참인 동안 반복

반복 횟수를 미리 모를 때, 또는 특정 조건이 될 때까지 반복해야 할 때 씁니다.

while_loop.py 
count = 1
while count <= 5:
    print(f"{count}번 반복")
    count += 1

print("---")

# break — 반복 강제 종료
n = 0
while True:
    n += 1
    if n == 3:
        break
print(f"break로 탈출! n = {n}")

# continue — 이번 반복만 건너뜀
for i in range(1, 6):
    if i % 2 == 0:
        continue           # 짝수는 건너뜀
    print(i, end=" ")

# 기본값 매개변수

🏗️ 설계도와 실체 — 클래스 class

클래스는 데이터(속성)와 기능(메서드)을 하나로 묶는 설계도입니다. 이 설계도로 찍어 낸 실제 객체를 인스턴스라고 합니다.

비유로 이해하기 — 클래스 = 스마트폰 설계도, 인스턴스 = 그 설계도로 만들어진 실제 스마트폰. 설계도는 하나지만, 스마트폰은 수백만 대를 찍어낼 수 있어요.

클래스 변수 vs 인스턴스 변수

이 두 가지 변수의 차이를 이해하는 것이 클래스 학습의 핵심입니다.

student_class.py 
class Student:
    # ── 클래스 변수: 모든 학생이 공유하는 정보 ──
    school = "파이썬 고등학교"
    count  = 0          # 학생 수 (전체 공유)

    # ── 생성자 메서드: 인스턴스가 만들어질 때 자동 실행 ──
    def __init__(self, name, grade, score):
        # 인스턴스 변수: 각 학생마다 독립된 정보
        self.name  = name
        self.grade = grade
        self.score = score
        Student.count += 1  # 학생 생성 시 카운터 증가

    # ── 일반 메서드 ──
    def study(self, hours):
        self.score += hours * 2
        print(f"{self.name}이(가) {hours}시간 공부 → 점수 {self.score}점")

    def get_info(self):
        print(f"[{Student.school}] {self.grade}학년 {self.name} — 점수: {self.score}")

    # ── 클래스 메서드: self 대신 cls 사용 ──
    @classmethod
    def get_count(cls):
        print(f"현재 등록된 학생 수: {cls.count}명")

인스턴스 생성 및 사용

상속 — 기존 클래스를 확장하기

기존 클래스(부모)를 물려받아 새로운 클래스(자식)를 만들 수 있습니다. 공통 기능은 부모에, 특수 기능은 자식에 넣으세요.

inheritance.py 
class Captain(Student):          # Student 상속
    def __init__(self, name, grade, score, team):
        super().__init__(name, grade, score)  # 부모 생성자 호출
        self.team = team                    # 추가 인스턴스 변수

    def get_info(self):                    # 메서드 오버라이딩
        super().get_info()
        print(f"  └ 소속 팀: {self.team} (팀장)")

cap = Captain("박지훈", 3, 90, "알고리즘반")
cap.get_info()
Student.get_count()

  └ 소속 팀: 알고리즘반 (팀장)
